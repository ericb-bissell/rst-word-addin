RST Word Add-in Help

Getting Started

What This Add-in Does

Convert your Word documents to reStructuredText (RST) format for use with Sphinx, Read the Docs, and other documentation systems.

  • Preview – See your document as RST in real-time
  • Copy – Copy RST text to clipboard
  • Export – Download RST file with images as ZIP

Quick Start

  1. Open any Word document
  2. Click RST Preview in the ribbon
  3. The preview panel shows your document as RST
  4. Click Copy or Export

Export Package

When you export, you receive a ZIP file containing:

your-document.zip
├── document.rst    (your converted document)
└── images/         (extracted images)
    ├── image_001.png
    └── figure_002.png

Extract the ZIP into your Sphinx project's source directory.

Buttons Reference

Refresh Update the preview manually
Copy Copy RST text to clipboard
Export Download ZIP with RST + images

Formatting Reference

Text Formatting

Bold **text**
Italic *text*
Code (monospace font) ``text``

Tip: Create a "Code" character style with Consolas font for inline code.

Headings

Heading 1 ==== with overline
Heading 2 ==== underline
Heading 3 ---- underline
Heading 4 ~~~~ underline
Heading 5 ^^^^ underline
Heading 6 """" underline

Lists

Bulleted lists:

- First item
- Second item
  - Nested item

Numbered lists:

1. First item
2. Second item
3. Third item

Links

Hyperlinks in Word convert to:

`Link Text <https://example.com>`__

Images

Images convert to the .. image:: directive:

.. image:: images/photo.png
   :alt: Description
   :width: 400px
   :align: center

Tip: Add alt text to images (right-click → Edit Alt Text) for better accessibility.

Figures (Images with Captions)

Use References → Insert Caption to add a caption:

.. figure:: images/diagram.png
   :align: center

   Figure 1: System overview

Tables

Word tables convert to RST grid tables:

+-------+-------+
| Col 1 | Col 2 |
+=======+=======+
| A     | B     |
+-------+-------+

Add a caption with References → Insert Caption → "Table".

Table of Contents

Word TOC fields convert to:

.. contents:: Table of Contents
   :depth: 3

Unsupported Formatting

The following have no RST equivalent:

  • Underline
  • Strikethrough
  • Text color / highlight
  • Font changes

Tip: Use custom styles like rst_note or rst_warning instead of colors to highlight important text.

Custom RST Directive Styles

How It Works

Word paragraph styles starting with rst_ convert to RST directives:

rst_note .. note::
rst_warning .. warning::
rst_code-block .. code-block::
rst_need .. need::

Creating a Custom Style

  1. Go to Home tab → Styles panel
  2. Click the small arrow to open Styles pane
  3. Click New Style
  4. Name it starting with rst_ (e.g., rst_note)
  5. Set Style type to "Paragraph"
  6. Click OK

Tip: Add a colored background or left border to make directive content visually distinct.

Simple Directive

Apply rst_note style to a paragraph:

Remember to save your work!

Output:

.. note::

   Remember to save your work!

Directive with Argument

Put the argument in square brackets on the first line:

[python]
def hello():
    print("Hello!")

Output (with rst_code-block style):

.. code-block:: python

   def hello():
       print("Hello!")

Directive with Options

Add options using :option: value format:

[python]
:linenos:
:caption: Example Code
def greet(name):
    print(f"Hi, {name}!")

Output:

.. code-block:: python
   :linenos:
   :caption: Example Code

   def greet(name):
       print(f"Hi, {name}!")

Sphinx-Needs Requirements

Style: rst_req

:id: REQ-001
:title: User Login
:status: open
The system shall authenticate users.

Output:

.. req::
   :id: REQ-001
   :title: User Login
   :status: open

   The system shall authenticate users.

Recommended Styles

rst_note Important notes
rst_warning Warning messages
rst_tip Helpful tips
rst_code-block Code with syntax highlighting
rst_req Requirements (sphinx-needs)
rst_todo TODO items

Troubleshooting

Preview Not Updating

  • Click the Refresh button to manually update
  • Check if the document has unsaved changes
  • Try closing and reopening the preview panel

Images Missing in Export

  • Ensure images are inserted as inline pictures (not floating/text wrap)
  • Very large images may take time to process
  • Try reducing image size before inserting

Custom Styles Not Converting

  • Style name must start with rst_ (lowercase)
  • Ensure the style is applied to the entire paragraph
  • Check for typos in the style name
  • Style type must be "Paragraph" (not "Character")

Export Download Fails

  • Check your browser's download settings
  • Ensure pop-ups are not blocked for Office.com
  • Try a different browser (Edge, Chrome, Firefox)
  • Check if you have enough disk space

Tables Look Wrong

  • Avoid heavily merged cells – simplify the table structure
  • Nested tables are not supported
  • Very wide tables may need manual adjustment in RST

Add-in Not Loading

  • Clear browser cache and reload the page
  • Check that JavaScript is enabled
  • Try sideloading the manifest again
  • Ensure you're using a supported browser

Getting More Help

If you're still having issues:

  • Check the browser console for error messages (F12 → Console)
  • Report issues on the project's GitHub repository
